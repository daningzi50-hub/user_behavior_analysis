# 基于淘宝亿级行为数据的电商用户分析实践

## 项目背景
基于淘宝用户行为数据集进行分析实践，原始数据约1亿条，
通过随机抽样10万用户（约1000万条）进行分析。
原始数据获取链接：https://tianchi.aliyun.com/dataset/649

## 技术栈
- Python（数据抽样）
- MySQL + Navicat（数据清洗与分析）
- Tableau（数据可视化）

## 数据预处理
- 随机抽取10万用户，保留完整行为记录
- 去除NULL值
- 去重（删除7条重复数据）
- 去除异常时间数据（删除5303行）
- 删除时间戳小于0的数据（删除93行）
- 最终保留数据约1008万条

## 分析模块
1. **用户行为分析** — PV/UV、每日趋势
2. **留存率分析** — 次日留存、3日留存
3. **流失率分析** — 最终活跃日期分布
4. **行为时段分析** — 各日期各小时行为分布
5. **转化漏斗分析** — 8条购买路径分析
6. **RFM用户分层** — 价值/保持/发展/挽留用户
7. **商品/品类分析** — 浏览量排名、转化率

## 转化率说明
由于数据采集时间较短，部分用户存在无浏览记录但有
购买行为的情况，转化率计算采用去重总用户数作为分母
以保证结果可解释性。
